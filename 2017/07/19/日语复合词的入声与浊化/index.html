日语复合词中的促音与连浊

发表于 2017-07-19 | 分类于 日语

在记忆日语单词发音的时候，复合词的发音是是个很让人头痛的地方，下面是我从网上各处搜罗整理的一些规律，和我平时积累的一些内容，希望对大家能有所帮助。

关于前字尾音的送气音

你可以理解为「有浊音现象的发音在未浊化的时候都是送气音」，为什么呢？正因为有了送气这个过程，才有了顿一下不送气的行为，而顿一下不送气，其实就是浊化了。因此，か、さ、た、は行的假名都是送气音。

而很多复合词里，如果两个组成部分链接的发音都是送气音，说起来的时候就需要一直向外呼气，是很不舒服的（或者说容易岔气），所以才有了前一部分尾音发生浊化的现象，让你顿一下，好留着气息把词说完。

因此有规律：
当前字以「き、く、ち、つ、ん」结束，后字以「か、さ、た、は」行的假名开头的时候，后字的首音发生浊化。

关于后字首音的浊化

对于复合词中后字首音是送气音的情况，有时候这个送气音也会发生浊化，或许当初发生浊化也是出于方便发音的考虑，但现在日语的发音经过一代代的改变，各种方言之间互相影响，已经彻底混乱了。比如山崎，受日本东西部方言的影响，东边的叫「やまざき」，西边的叫「やまさき」，又比如赤地，有的叫「あかち」有的叫「あかじ」，还有大岛，有的叫「おおしま」有的叫「おおじま」，日本人自己也是很混乱的。

因此这里也只是整理了少数有规律的，实际在背单词的时候，建议直接多背几遍吧。

有关音读的几个规律

1.当两个送气音相遇时，就会将前一个送气音变为「っ」，例如：

学（がく）＋　校（こう）→　学校（がっこう）
日（にち）＋　記（き）　→　日記（にっき）
一（いち）＋　冊（さつ）→　一冊（いっさつ）
察（さつ）＋　する　　　→　察する(さっする)
結（けつ）＋　婚（こん）→　結婚（けっこん）
活（かつ）＋　気（き）　→　活気（かっき）
列（れつ）＋　島（とう）→　列島（れっとう）
達（たつ）＋　者（しゃ）→　達者（たっしゃ）
国（こく）＋　家（か）　→　国家（こっか）
特（とく）＋　権（けん）→　特権（とっけん）

2.前一个送气音变「っ」后，后一个送气音如为は行则变为半浊音。例如：

失（しつ）＋　敗（はい）→　失敗（しっぱい）
一（いち）＋　匹（ひき）→　一匹（いっぴき）
烈（れつ）＋　風（ふう）→　烈風（れっぷう）
鉄（てつ）＋　片（へん）→　鉄片（てっぺん）
一（いち）＋　本（ほん）→　一本（いっぽん）

3.前一个汉字以「ん」结尾，后一汉字发音的第一假名若是は行开头的变为ぱ行， 也有少数变成ば行的。例如：

先（せん）＋　輩（はい）→　先輩（せんぱい）
心（しん）＋　配（はい）→　心配（しんぱい）
神（しん）＋　秘（ひ）　→　神秘（しんぴ）
何（なん）＋　分（ふん）→　何分（なんぷん）
南（なん）＋　北（ほく）→　南北（なんぼく）
藍（らん）＋　本（ほん）→　藍本（らんぽん）

4.前一个汉字以「ん」结尾，后一个汉字若是以元音开头的，变为「な」行假名或者「ま」行假名。

反（はん）＋　応（おう）→　反応（はんのう）
三（さん）＋　位（い）　→　三位（さんみ）

有关训读的几个规律

1.か、さ、た、は即送气音开头的单词，接在其他词后构成复合词时，发生浊音化。 例如：

物（もの）　＋　語り（かたり）→　物語（ものがたり）
鼻（はな）　＋　血（ち）　　　→　鼻血（はなぢ）
足（あし）　＋　取り（とり　）→　足取り（あしどり）步伐，步调；踪迹。
昔（むかし）＋　話（はなし）　→　昔話（むかしばなし） 旧话老话；传说，故事。
人（ひと）　＋　人（ひと）　　→　人々（ひとびと）

2.前一个汉字最后一个假名在え段的，变为同一行的あ段假名。例如：

雨（あめ）　＋　水（みず）　　→　雨水（あまみず）
稲（いね）　＋　光（ひかり）　→　稲光（いなびかり）

3.例外的情况：

但本身含浊音的训读词，不发生连浊。例如：
紙屑（かみくず）　　　大風（おおかぜ）
动词与动词或动词与宾语的复合不发生连浊。例如：
差し引く（さしひく）　読み書き（よみかき）　飯炊き（めしたき）
