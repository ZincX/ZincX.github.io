合租房的室友

发表于 2017-01-09 | 分类于 日常

上个月27号的时候，突发奇想准备换个住处。已经忘了打开租房APP的契机是什么了，只记得打开之后，看到稍远一些的地方，房租比当时住的要便宜很多。当时住的是3090/月，加上一年3000多的服务费，还有每个月的水电，均摊下来，每个月也有3500左右了，而查到的房间，便宜的有2000出头的。一个人也没有多大必要住太大的房子，当时住的朝南，双人床，有阳台，对于孑然一人的我来说，实在是有太大的空间浪费掉了。与其为一片片空地付房租，不如换到一个空间利用率更高的地方。于是当天选好了房源，预约第二天看房，看房之后，便当场签了合同，当晚便把东西搬个大半，第三天又搬了些零碎的东西，第四天中午看了下之前的房子没有大碍，就算是彻底搬了过来。

感觉搬来这里已经是一个月前的事了，其实也就一周多的时间。

1号房间住了一个妹子，有时男朋友会过来，我搬来的那几天，她整夜整夜地在电话里跟男朋友吵架，每天都吵到一两点，墙的隔音又很差，吵架的内容能听个差不多。一天天的，慢慢也就听懂了她在跟她男朋友吵什么。刚来的那几天，是在吵她男朋友跟其他女生搞暧昧，之后吵则是因为男的答应的跟她一起过元旦，她推掉了一切的聚会，买了一堆他喜欢吃的东西，结果男的临时找了个借口自己玩去了，再之后，则是女的想跟男的结婚，于是春节想去见男方的家长，结果他也推掉了，说自己小姨不喜欢她，春节就不要去了云云。偶然的一次，我进门的时候瞥见过她的男朋友，是那种很呆滞的长相，一脸茫然无神，没想到却能把隔壁的妹子忽悠地团团转，后来听到妹子开心地说准备今年结婚，打算去美国生孩子了。真的这样的男朋友能留到结婚么？

我想租的其实是3号房间，比我的房间贵了100，却大了整整一圈，住的是个男的，但是前天早晨还听到有女的来找他，那时我被吵醒了，看了一眼表，5:10，那时地铁还没通，她是怎么来的？搞不懂，怎么想都搞不懂她是怎么来的，下夜班？之后听到他们洗漱的声音，然后是睡觉，然后是行李箱的声音。洗手间之前放着男的一瓶洗发露，一管洗面奶，昨天已经空了，没有这个男的一点痕迹，房间也没再亮过灯，回家了？这也太早了，搬家了？网上显示房子还是出租状态。

5号房间住着一个妹子，通过微信群看过她的朋友圈，洗手间经常有两副牙刷牙膏，但是只有一份洗面奶洗发水，第二套牙刷牙膏可能是一个短暂居住的人的，是男是女还不清楚，也是在前几天，第二份牙刷的杯子里有了第三把牙刷，是一把电动牙刷，所以可能这三个都是妹子，要是三人里有一个男生，就觉得有些怪怪的了。这个妹子貌似是个设计类的工作，有专门的画板，晚上不加班，总是很早下班，在宿舍里打守望先锋打到很晚。虽然微信里有看到她的这些信息，不过房间里却一直没有见过她。

冰箱里现在都是我和1号房间妹子的，3号房没有往冰箱放过东西，5号房的妹子放了一桶牛奶，仅此而已。

1号房间有独立的卫生间，所以外面的卫生间是我与3、5号房间共用，我留意到上周末之后，马桶边上多了一卷纸巾，放在一个新粘上去的防水盒子里。我搞不懂为什么他们都喜欢把纸巾放在公共的区域，之前在广州的时候室友赵Y也喜欢这样，这样造成的麻烦就是我得在我的洗漱用具那里放一小包纸巾，还得记得随时更换，好让他们相信我没有用他们的纸巾。我当然不会用他们的，在这方面我是极其讲究卫生的，所以我也买了自己用的马桶垫——我留意到那卷纸巾用得特别快，下面纸篓里用过的纸也不像是擦屁股用的，所以只可能是：用来擦马桶垫的了。想不通为什么宁愿一次次用纸，还不保证能够擦得很干净，而不愿买一个马桶垫呢？——也因此我猜测这卷纸是3号房间男生的，5号房间的妹子肯定会买马桶垫的，显而易见，这样一来，我倒也不用担心他担心我用他的纸了，一个心大到用纸擦马桶垫的人，肯定不会考虑我是否会用他的纸的。
